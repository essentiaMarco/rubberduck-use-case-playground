# Demo projects hub

RubberDuck-styled control plane for lab demos in this repo.

## Getting started

1. **Git clone** `rubberduck-use-case-playground`
2. **Open the hub** — `start-hub.bat` (Windows) or `./start-hub.sh` (Linux/macOS) → http://127.0.0.1:5055/
3. **Welcome** — choose a demo
4. **Launch** — live setup console + health diagram
5. **Connect RubberDuck MCP** — **Connect MCP** tab in the hub (copy `mcp.json`, health check, index). See also [SETUP.md](../SETUP.md).
6. **Generate prompts** — UC-01…UC-10, **What to expect**, copy for Cursor / Claude

Demos are child folders with a `setup_and_run.py` (currently: `rubberduck_pizzeria-demoapp`).
