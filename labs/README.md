# Labs directory

Each `ucXX_*` folder is a **runnable mini-project** for one RubberDuck workflow.

## Run any lab

```bash
python scripts/run-lab.py --uc 02 --verify
python scripts/run-lab.py --uc 02 --start-server   # live API
```

See [manifest.json](manifest.json) for machine-readable lab metadata.
